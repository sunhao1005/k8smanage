# k8smanage 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为单机 k3s 小集群做一个「集群管理 + 基础监控」一体的 Web 工具，打成单个 Go 镜像（前端 embed 进二进制），复用 k3s 自带 metrics-server，新增常驻内存 ≈120MB。

**Architecture:** 一个 Go 进程同时承担：① 用 controller-runtime informer 缓存 k8s 对象状态（读不打 apiserver）；② 用 gopsutil 读宿主机指标 + 读 metrics-server 拿 Pod 指标，落 SQLite 环形存储；③ chi REST + WebSocket 提供聚合 API；④ embed 的 React 控制台。所有采集/存储/管理在一个容器内，metrics-server 保持独立作为兜底。

**Tech Stack:** Go 1.22 / client-go / sigs.k8s.io/controller-runtime / k8s.io/metrics / shirou/gopsutil/v4 / modernc.org/sqlite（纯 Go 无 CGO）/ go-chi/chi / coder/websocket / Vite + React + TS + uPlot。

---

## 设计约束（贯穿全程，违反即返工）

1. **读操作只读 informer 缓存**，禁止在请求路径里直连 apiserver List/Get。
2. **单镜像无 CGO**：SQLite 用 `modernc.org/sqlite`，`CGO_ENABLED=0` 编译，落 distroless。
3. **主机采集走接口 `NodeCollector`**：今天进程内 gopsutil 实现；多节点时换 agent 实现，API/前端不动。
4. **存储走接口 `MetricStore`**：今天 SQLite 实现；将来要长期历史可换 VictoriaMetrics 实现，前端不动。
5. **凭据不入库**：kubeconfig / webhook 密钥从环境变量或挂载读，不写进仓库、不进镜像层。
6. **TDD + 频繁提交**：每个 Task 一个可独立验证的提交。中文 commit + conventional 前缀。

## 核心接口与数据模型（先定契约，后面 Task 都引用这里）

```go
// internal/metrics/store.go
package metrics

import (
	"context"
	"time"
)

// TargetKind 区分样本归属：节点 or Pod。
type TargetKind string

const (
	TargetNode TargetKind = "node"
	TargetPod  TargetKind = "pod"
)

// Sample 一条采集样本（一个 target 一个时刻的一组指标）。
type Sample struct {
	Kind   TargetKind
	Target string    // 节点名 或 "ns/pod"
	TS     time.Time
	CPU    float64   // 已用核数（cores）
	MemUse uint64    // 已用内存 bytes
	MemTot uint64    // 总内存 bytes（Pod 为 limit，0 表示无 limit）
	DiskUse uint64   // 仅节点：根分区已用 bytes
	DiskTot uint64   // 仅节点
	NetRx  uint64    // 仅节点：累计接收 bytes
	NetTx  uint64    // 仅节点：累计发送 bytes
	Load1  float64   // 仅节点
}

// Point 区间查询返回的单点（按 metric 投影后）。
type Point struct {
	TS    time.Time
	Value float64
}

// MetricStore 抽象存储；默认 SQLite 实现，可换 VictoriaMetrics。
type MetricStore interface {
	Write(ctx context.Context, s []Sample) error
	// Query 取某 target 某 metric 在 [from,to] 的序列；metric ∈ {cpu,mem,disk,net_rx,net_tx,load1}
	Query(ctx context.Context, kind TargetKind, target, metric string, from, to time.Time) ([]Point, error)
	// Latest 取某 target 最近一条样本（总览用）。
	Latest(ctx context.Context, kind TargetKind, target string) (Sample, bool, error)
	// Prune 删除早于 cutoff 的样本（环形保留）。
	Prune(ctx context.Context, cutoff time.Time) (int64, error)
	Close() error
}
```

```go
// internal/alert/rule.go
package alert

// Comparator 阈值方向。
type Comparator string

const (
	GT Comparator = ">"
	LT Comparator = "<"
)

// AlertRule 一条阈值规则。
type AlertRule struct {
	ID       string     // uuid
	Name     string
	Kind     string     // "node" | "pod"
	Target   string     // 为空=匹配全部该 Kind 的 target
	Metric   string     // cpu|mem|disk|load1（mem/disk 按使用率 0..1）
	Cmp      Comparator
	Threshold float64
	ForSec   int        // 持续多少秒才 firing
	Enabled  bool
}

// AlertState 规则当前状态。
type AlertState string

const (
	StateOK      AlertState = "ok"
	StatePending AlertState = "pending"
	StateFiring  AlertState = "firing"
)
```

**SQLite schema**（`store_sqlite.go` 建表）：

```sql
CREATE TABLE IF NOT EXISTS samples (
  kind TEXT, target TEXT, ts INTEGER,         -- ts=unix 秒
  cpu REAL, mem_use INTEGER, mem_tot INTEGER,
  disk_use INTEGER, disk_tot INTEGER,
  net_rx INTEGER, net_tx INTEGER, load1 REAL
);
CREATE INDEX IF NOT EXISTS idx_samples ON samples(kind, target, ts);
CREATE TABLE IF NOT EXISTS alert_rules (
  id TEXT PRIMARY KEY, name TEXT, kind TEXT, target TEXT,
  metric TEXT, cmp TEXT, threshold REAL, for_sec INTEGER, enabled INTEGER
);
```

```go
package config

import (
	"os"
	"strconv"
)

type Config struct {
	Addr         string
	IntervalSec  int
	RetentionSec int
	DBPath       string
	NodeName     string // 从 downward API NODE_NAME 注入
	AlertWebhook string
}

func Load() Config {
	return Config{
		Addr:         env("K8SM_ADDR", ":8080"),
		IntervalSec:  envInt("K8SM_INTERVAL_SEC", 15),
		RetentionSec: envInt("K8SM_RETENTION_SEC", 7*24*3600),
		DBPath:       env("K8SM_DB_PATH", "/data/k8smanage.db"),
		NodeName:     env("NODE_NAME", ""),
		AlertWebhook: env("K8SM_ALERT_WEBHOOK", ""),
	}
}

func env(k, d string) string { if v := os.Getenv(k); v != "" { return v }; return d }
func envInt(k string, d int) int { if v := os.Getenv(k); v != "" { if n, err := strconv.Atoi(v); err == nil { return n } }; return d }
```

```go
	"context"

func TestSQLiteWriteQueryLatest(t *testing.T) {
	ctx := context.Background()
	s, err := NewSQLiteStore(":memory:")
	require.NoError(t, err)
	defer s.Close()

	base := time.Unix(1_700_000_000, 0)
	require.NoError(t, s.Write(ctx, []Sample{
		{Kind: TargetNode, Target: "n1", TS: base, CPU: 0.5, MemUse: 100, MemTot: 200},
		{Kind: TargetNode, Target: "n1", TS: base.Add(15 * time.Second), CPU: 0.7, MemUse: 150, MemTot: 200},

	require.Len(t, pts, 2)
	require.Equal(t, 0.5, pts[0].Value)

	last, ok, err := s.Latest(ctx, TargetNode, "n1")
	require.NoError(t, err)
	require.True(t, ok)
	require.Equal(t, 0.7, last.CPU)
}

func TestSQLitePrune(t *testing.T) {

	old := time.Unix(1_000, 0)

	require.NoError(t, err)
```

## 风险与缓解（已在前期评审确认，执行时复核）

1. **host /proc 读取**：容器需挂宿主 `/proc /sys` 只读 + `HOST_PROC/HOST_SYS`（node-exporter 同款）——在 5.4 清单落实。
2. **SQLite 持久化**：WAL 模式 + hostPath 卷，避免重启丢历史。
3. **多节点边界**：当前 NodeCollector 只采本机；多节点改 agent 实现，API/前端不动。
4. **提权 pod**：`hostPID` + host 挂载，RBAC 最小化，仅授必要读写。
5. **metrics-server 依赖**：k3s 自带；若被禁用则 Pod 指标缺失（节点指标不受影响），文档提示保留 metrics-server。
